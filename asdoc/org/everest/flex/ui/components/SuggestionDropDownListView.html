Classpublic class SuggestionDropDownListView
InheritanceSuggestionDropDownListView FilterComboBox spark.components.ComboBox

This is a complete independent view that its own presenter and data model. The view loads a collection from the configured url and uses it to provide an auto complete drop-down box. It can be used as part of a form to filter a collection or to edit/create a new member.
