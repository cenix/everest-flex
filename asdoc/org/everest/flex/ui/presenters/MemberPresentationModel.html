Classpublic class MemberPresentationModel
InheritanceMemberPresentationModel flash.events.EventDispatcher

Contains the presentation logic and data for the REST member view. This class contains methods to access and manipultate the member data model. It can be extended by an everest implementation to add additional featrues for each member and its view.
