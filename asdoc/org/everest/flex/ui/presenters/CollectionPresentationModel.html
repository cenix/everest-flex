Classpublic class CollectionPresentationModel
InheritanceCollectionPresentationModel flash.events.EventDispatcher

Contains the presentation logic and data for the REST collection view. This class contains methods to access and manipultate the collection data model. It can be extended by an everest implementation to add additional featrues.

See also
