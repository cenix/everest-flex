Packageorg.everest.flex.ui.views
Classpublic class CollectionView
InheritanceCollectionView spark.components.VGroup
Implements mx.binding.IBindingClient, mx.core.IStateClient2

This is a configruable component that all everest collection views can simply extend.
  • In the top it contains an optional filter form. (configurable filterFormContent components)
  • In the center adatagrid with the collection items. (configurabel columns through method: collectionGridColumns())
  • In the bottom left buttons to create or delete items are shown.
  • In the bottom on the right custom components can be added. (configurable footerContent components)

Default MXML PropertymxmlContent
