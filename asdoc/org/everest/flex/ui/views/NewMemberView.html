Classpublic class NewMemberView
InheritanceNewMemberView ResizableTitleWindow spark.components.TitleWindow
Implements mx.binding.IBindingClient

This is a template that can be used to create the user interface for creating new members to be added to a collection. The custom NewMemberViews can be directly injected into the collection view. Then a button will be shown that opens this window to allow the user to add new members to the specific collection view.
