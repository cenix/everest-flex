Classpublic class MemberView
InheritanceMemberView spark.components.VGroup
Implements mx.binding.IBindingClient, mx.core.IStateClient2

This is a component that all member views of an everest flex application inherit from. The main content will go into the content section. If components are set to the editableContent section an edit button will be shown. The edit view will be loaded and the save button will trigger a model.submit action. The additionalContent section goes below and it not intended to be editable.
