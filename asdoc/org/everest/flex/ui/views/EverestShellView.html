Classpublic class EverestShellView
InheritanceEverestShellView spark.components.VGroup
Implements mx.binding.IBindingClient

This view is the context in which every everest flex application is running. In the main application class the ShellEventMap needs to be initialzed with a custom configuration. Then the EverestShellView has to be added to the screen. Everything around can be customized depending on the application needs.
