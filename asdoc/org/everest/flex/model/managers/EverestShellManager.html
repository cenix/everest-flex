Classpublic class EverestShellManager
InheritanceEverestShellManager flash.events.EventDispatcher

In combination with the shell event map this class represents the cental everest application controller. It handles: - application initialization and configuration - serialization and deserialization - url construction - feed and document model - loading of modules
