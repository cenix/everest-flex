Classpublic class EverestConfiguration
InheritanceEverestConfiguration Object

An everest application must instantiate and inject this configuration in the EverestEventMap. The configuration includes definitions for mapping the server data model to the client data model and back. It furthermore defines a list of flex application modules and for which content type they shall be loaded.
